## HWInfo-Log-Viewer 介绍

[English](README.md)


HWInfo-Log-Viewer是一个基于`Tauri`和`Vue3`的跨平台应用程序，通过可视化HWiNFO日志记录生成的CSV
日志文件，提供了一个用户友好的界面，允许用户轻松浏览和分析传感器数据。

## i18n（仅 UI）

- 可在 **系统设置 → 偏好设置 → 语言** 中切换界面语言。
- 侧边栏中的传感器/分组名称优先来自 CSV 表头或后端分组数据，**不会被 UI i18n 覆盖翻译**。

- 英文示例：`src-tauri/data/1_EN.CSV`

## 功能规划

- [x] 支持上传CSV文件，并正确可视化数据为折线图
- [x] 支持数据筛选和排序
- [x] 数值显示等易用性提升
- [x] 支持选择并更换配置文件
- [x] 支持坐标轴的缩放，最大最小值虚线
- [ ] 支持多种图表类型（折线图、柱状图、饼图等
- [ ] 支持多数据源合并分析
